Worked performed:

  • Created new Debian live install images git repository on github:
  • ssh://git@github.com/marcilr/debian-live-install-images
  • Checked new git repository out to:

    ~/git/os/debian/live-install-images

  • Download Debian Live install image DVD/USB amd64 image:

    open Debian Live install images:
    https://www.debian.org/CD/live/

    click DVD/USB amd64 (bottom left)
    https://cdimage.debian.org/debian-cd/current-live/amd64/bt-hybrid/

    select latest debian live xfce torrent (2022-03-26 17:53, 49K):
    click debian-live-11.3.0-amd64-xfce.iso.torrent

    Install lightweight BitTorrent client:
    $ sudo aptitude install transmission-gtk transmission-cli

    Execute transmission-gtk application:
    $ transmission-gtk

    Download debian-live-11.3.0-amd64-xfce.iso.torrent
    In transmission-gtk:

    click File
    click Open URL...
    select ~/Downloads/debian-live-11.3.0-amd64-xfce.iso.torrent
    click Open

    At Torrent Options
    Torrent file: debian-live-11.3.0-amd64-xfce.iso.torrent (2.59GB)
    Destination Folder: Downloads
    click Open

    Download took ~3 minutes

    Verify and take sha256sum of torrent file:
    $ cd ~/Downloads/
    $ type sha256sum
    sha256sum is hashed (/usr/bin/sha256sum)
    $ sha256sum debian-live-11.3.0-amd64-xfce.iso
    713443a1326deaf03343fef9cb9346d5a1b2f5634a138540ae549a7b89b3adff debian-live-11.3.0-amd64-xfce.iso

    The debian-live-11.3.0-amd64-xfce.iso sha256sum matched that listed in:
    https://cdimage.debian.org/debian-cd/current-live/amd64/bt-hybrid/SHA256SUMS

    The download looks good.
